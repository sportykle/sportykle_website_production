# Sportykle Website

Premium multi-page SaaS marketing website for Sportykle — a cloud-based Sports, Fitness, Wellness, and Multi-Center Management Platform.

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS v4
- Framer Motion
- React Router v7
- Vite

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Features

- Dark & light mode
- Fully responsive (mobile-first)
- SEO optimized meta tags
- Animated statistics & transitions
- Dashboard & mobile app mockups
- Conversion-focused CTAs throughout
